Documentação do pacote tabatu
=============================

O pacote tabatu oferece uma API para obter de forma simples quantidades relevantes sobre tábuas atuárias, isto é,
tábuas biométricas, taxas de juros, comutações e fluxos.

Instalação
----------

Para usar o pacote, é necessário realizar a instalação via pip:

.. code-block:: console

   # pip install -e .
