Documentação do pacote tabatu
=============================

O pacote tabatu é uma interface em python com códigos em C++ para tábuas atuariais. São fornecidos métodos e classes
para lidar com probabilidades referentes a tábuas biométricas, taxas de juros e comutações.

Instalação
----------

Para usar o pacote, é necessário realizar a instalação via pip:

.. code-block:: console

   # pip install -e .
